プログラム復習コンテンツ

pushMatrix(),popMatrix関数

サンプルコードの結果

サンプルコード

　

　　 void setup(){

　　 　size(120, 120);

　　 ｝

　　

　　 void draw(){

　　 　pushMatrix();

　　 　translate(mouseX, mouseY);

　　 　rect(0, 0, 30, 30);

　　 　popMatrix();

　　 　translate(35, 10);

　　 　rect(0, 0, 15, 15);

　　 }　　

説明文

　pushMatrix(),popMatrix()関数は座標系を復元させる関数。

　pushMatrix()とpopMatrix()は常にペアで使用する。

　pushMatrix()時点での座標系を記録し、popMatrix()で記録した座標系に復元する。

　例では、pushMatrix()時点では、座標系は基本の（0，0）となっている。

　1つ目の四角形はmouseX,mouseY変数によって座標系が毎回変わるが、2つ目の四角形は、popMatrix()によって座標系が（0，0）に変更されているため、mouseX,mouseY変数の影響を受けずに常に同じ位置に表示される。
